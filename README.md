# 🎫 티켓팅 타이머 - Chrome 확장 프로그램

공연 티켓팅을 위한 정확한 시간 동기화 및 탭별 타이머 확장 프로그램입니다.

## ✨ 주요 기능

- **정확한 표준시 동기화**: WorldTimeAPI를 통해 KST 표준시와 동기화
- **탭별 독립 타이머**: 각 탭마다 서로 다른 공연의 오픈 시각 설정 가능
- **실시간 카운트다운**: 남은 시간을 실시간으로 표시
- **자동 새로고침**: 설정 시각에 자동으로 페이지 새로고침 (선택 사항)
- **알림**: 티켓 오픈 시각에 브라우저 알림 + 탭 자동 활성화
- **배지 표시**: 각 탭에 남은 시간을 배지로 표시

## 📦 설치 방법

### 개발자 모드로 설치

1. Chrome 브라우저를 엽니다
2. 주소창에 `chrome://extensions/` 입력
3. 오른쪽 상단의 **개발자 모드** 토글을 켭니다
4. **압축해제된 확장 프로그램을 로드합니다** 버튼 클릭
5. `ticket-timer` 폴더를 선택합니다

## 🚀 사용 방법

### 1. 타이머 설정하기

1. 티켓팅할 사이트 탭을 엽니다 (예: 멜론티켓, 인터파크 등)
2. 확장 프로그램 아이콘을 클릭합니다
3. 다음 정보를 입력합니다:
   - **공연명**: 예) 아이유 콘서트
   - **날짜**: 티켓 오픈 날짜
   - **시각**: 티켓 오픈 시각 (초 단위까지 정확히)
   - **자동 새로고침**: 체크하면 정각에 자동으로 F5 누름
4. **이 탭에 타이머 설정** 버튼 클릭

### 2. 여러 공연 동시 준비

- 각 티켓팅 사이트를 별도 탭으로 엽니다
- 각 탭에서 확장 프로그램을 열어 독립적으로 타이머를 설정합니다
- 각 탭의 배지에 남은 시간이 표시됩니다

### 3. 타이머 관리

- 확장 프로그램 팝업 하단에서 설정된 모든 타이머를 확인할 수 있습니다
- **탭으로 이동**: 해당 티켓팅 사이트로 바로 이동
- **삭제**: 타이머 제거

### 4. 티켓팅 시각 도래 시

- 브라우저 알림이 표시됩니다
- 해당 탭이 자동으로 활성화됩니다
- 자동 새로고침을 켰다면 페이지가 새로고침됩니다

## 🎯 팁

- **시간 동기화**: 팝업을 열 때마다 자동으로 표준시와 동기화됩니다
- **정확도**: 네트워크 지연까지 보정하여 밀리초 단위로 정확합니다
- **배지**: 탭의 배지를 보면 남은 시간을 한눈에 확인할 수 있습니다
  - `3d` = 3일 남음
  - `2h` = 2시간 남음
  - `45m` = 45분 남음
  - `30s` = 30초 남음
  - `!` = 시간 도래!

## 🔧 기술 스택

- Chrome Extension Manifest V3
- Vanilla JavaScript (프레임워크 없음)
- WorldTimeAPI (시간 동기화)
- Chrome Storage API (데이터 저장)
- Chrome Alarms & Notifications API

## ⚠️ 주의사항

- 이 확장 프로그램은 **시간 확인 및 알림**만 제공합니다
- 자동 클릭, 자동 구매 등의 매크로 기능은 없습니다
- 티켓 구매는 직접 수동으로 진행해야 합니다
- 일부 티켓팅 사이트는 봇 방지를 위해 자동 새로고침을 감지할 수 있습니다

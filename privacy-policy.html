Privacy Policy

Last Updated: January 28, 2025

1. Overview

Ticketing Timer ("this Extension") prioritizes user privacy. This Privacy Policy explains what information is collected and how it is used.

2. Information We Collect

This Extension only stores the following information locally in your browser:

  • Timer settings you configure (event name, opening time)
  • Title and URL of tabs where timers are set
  • Auto-refresh preferences

3. Data Storage and Use

  • All data is stored only in Chrome's local storage
  • No personal information is transmitted to external servers
  • Collected information is used solely for providing timer functionality
  • We do not share information with third parties

4. External Services

This Extension uses the following public APIs for accurate time synchronization:

  • Google (www.google.com) - Server time from HTTP headers
  • Cloudflare (cloudflare.com) - Time synchronization API

These requests only retrieve time information and do not transmit personal data.

5. Permissions Usage

Permissions requested by this Extension and their purposes:

  • storage: To save timer settings locally
  • alarms: To provide notifications at set times
  • notifications: To display ticketing time alerts
  • tabs: To identify tabs where timers are set
  • host_permissions: To access time synchronization APIs

6. Data Deletion

Users can delete stored data at any time by:

  • Deleting individual timers within the Extension
  • Removing the Extension (automatically deletes all data)
  • Chrome Settings > Privacy and security > Clear browsing data

7. Security

This Extension strives to protect user data securely. All data is stored locally only, and uses encrypted HTTPS connections exclusively.

8. Children's Privacy

This Extension does not intentionally collect personal information from children under 13 years of age.

9. Privacy Policy Changes

This Privacy Policy may be updated as necessary. Significant changes will be announced through the Extension.

10. Contact

If you have questions or concerns about this Privacy Policy, please contact us through the Chrome Web Store support section.
